# Rebound
Personalized athlete injury recovery platform that helps users track, understand, and manage their rehabilitation journey.

[fill in your stack here — React Native? Swift? Web?]

## Core Features
- Interactive body diagram (color coded: red = bad, green = recovering)
- Injury setup profiler (how it happened, sport, date, doctor's diagnosis, symptoms)
- Daily recovery check-ins (pain score, symptom tracking)
- Photo-based recovery timeline with AI analysis (redness, bruising, swelling)
- PT rehab exercise tracker
- PT finder tool (insurance provider + zip code)
- Range of motion tracking via computer vision
- Recovery dashboard (charts, trends, photo timeline)

## Optional Features
- AI recovery coach (weekly summaries, pattern recognition — not a replacement for doctors/PTs)
- Reminders for daily symptom logging
- Voice input for accessibility
- Diet app integration
- Apple Health integration

## Notes
- Target users: athletes recovering from injuries
- Do not replace doctors or physical therapists — app is a tracking/assistant tool only
